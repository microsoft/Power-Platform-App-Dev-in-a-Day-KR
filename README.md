# 파워 플랫폼 앱 개발 핸즈온

이 핸즈온 랩 예제를 통해 여러분은 아래와 같은 결과를 얻을 수 있습니다.

- 파워 앱과 파워 오토메이트를 통해 한 회사의 인사관리 시스템을 고도화 시킬 수 있습니다.
- 파워 플랫폼 커스텀 커넥터를 이용해 다양한 API를 연결하고, 애저 API 관리자를 이용해 좀 더 향상된 보안 설정을 할 수 있습니다.
